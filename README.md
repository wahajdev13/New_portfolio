## **Color Palette & Typography**

**Color Palette Used:**

* **Primary Color:** `#7820db` (Vivid Purple) - Used for text highlights, hovers, and primary interactive elements
* **Secondary Color:** `#060635` (Dark Purple) - Used for navigation bar and secondary background elements
* **Accent Color:** `#indigo-400` (Light Indigo) - Used for hover states and interactive feedback
* **Background Color:** `#000026` (Dark Navy) - Used for main page background
* **Text Color:** `#ffffff` (White) - Used for primary text content

**Typography Scale:**

* **Font Family:** `"Poppins"` - Main font used throughout the portfolio
* **h1 Size:** `2.5rem` (40px) - Used for main page headings and hero titles
* **h2 Size:** `2rem` (32px) - Used for section headings
* **h3 Size:** `1.5rem` (24px) - Used for subsection headings
* **p Size:** `1rem` (16px) - Used for body text and paragraphs
